# eye-tracker
Bu proje, bilgisayar kamerası üzerinden gerçek zamanlı göz takibi yapmayı sağlayan bir yazılım çalışmasıdır. Üniversite projesi kapsamında geliştirilmektedir.

#Hedefler 
Yüksek performans, olabildiğince hızlı çalışması (yüksek fps).
Maksimum Doğruluk, mediapipe kullanarak göz çevresi ve göz bebeği hareketlerinin en düşük hata payıyla tespit edilmesi.
